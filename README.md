# WhatsApp Auto Sender

WhatsApp Auto Sender is a Python desktop application that sends personalized WhatsApp messages automatically using WhatsApp Web.

## Features

- Send messages to multiple contacts from CSV
- Personalized message using [nama]
- Optional image sending
- Tkinter GUI
- Selenium automation
- Microsoft Edge support

## Requirements

- Python 3.11+
- Microsoft Edge
- Edge WebDriver

Install dependencies

pip install -r requirements.txt

Run

python wa_sender_gui.py

CSV Format

nama,nomor,kirim
John,6281234567890,yes
Jane,6289876543210,yes
